== Description ==

Are you looking for a Countdown timer for Elementor or Short code supported? We are here to help you.You will love the simplicity of the plugin.
We make it for those who dont want to load junk assets.If you dont have enough time to maintain your Countdown then Ninja Countdown is the best option to choose.

It is that Countdown Plugin that you expected in your plugins bucket! It just needs 2 clicks to make a Beautiful New Countdown!!

= Why Ninja Countdown ? =

* **Fastest** Ninja Countdown Builder is the Fastest Countdown Builder.
* **Simplest** Simple to set up.
* **Lightweight** No un-used assets loads. Assets size is just 10kb.
* **Organized** The plugin is well Organized to set up a new Countdown.
* **Live Preview** You can edit and show live preview 
* **Elementor Widget** For the Elementor experts we have Elementor Widget too.
* **Multiple Designs** Finally we introduce Multiple Countdowns (Yes, In the same page, of course!!).
